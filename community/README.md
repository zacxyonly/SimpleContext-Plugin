# Community Plugins

Folder ini untuk plugin yang dikontribusikan komunitas.

## Cara Submit Plugin

1. Baca [CONTRIBUTING.md](../CONTRIBUTING.md)
2. Fork repo ini
3. Buat folder `plugin-nama-kamu/` di sini
4. Salin dari `../templates/plugin-starter/` dan sesuaikan
5. Buka Pull Request

## Panduan Penamaan

- Folder: `plugin-nama-kamu/` (kebab-case)
- File plugin: `nama_plugin.py` (snake_case)
- Class name: `NamaPlugin` (PascalCase)
- `name = "nama_plugin"` (snake_case, unik)
