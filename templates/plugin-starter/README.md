# plugin-nama-kamu

> Deskripsi singkat plugin ini.

## Cara Pasang

Salin `nama_plugin.py` ke folder `plugins/` di project SimpleContext kamu.

## Konfigurasi

```yaml
plugins:
  nama_plugin:
    enabled: true
    option_a: value
```

| Option | Type | Default | Deskripsi |
|--------|------|---------|-----------|
| `option_a` | string | `"default"` | Deskripsi opsi |
